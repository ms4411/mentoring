이 레포지토리는 우리 수업의 메인 허브 역할을 할 곳이에요. 눈에 보이는 예쁜 화면(Frontend)부터, 데이터를 처리하는 보이지 않는 서버(Backend/DB)까지! 우리가 앞으로 배우고 실습할 모든 코드들을 여기에 차곡차곡 쌓아나갈 예정입니다.

## 📅 전체 커리큘럼 (8주 로드맵)
매주 목요일, 이론 1시간과 실습 1시간으로 알차게 진행됩니다. 

- **1주차:** OT 및 개발 환경 세팅 (WSL, Git/GitHub 익숙해지기)
- **2주차:** [Frontend] HTML, CSS - 내 사이트 뼈대 잡고 예쁘게 꾸미기
- **3주차:** [Frontend] Node.js, TypeScript - 사이트에 움직임 더하기
- **4주차:** [Frontend] React + Vite - 여러 페이지 부드럽게 연결해보기
- **5주차:** [Backend] Express - 보이지 않는 서버와 API 만들기
- **6주차:** [Database] MySQL - 우리 서비스 데이터 안전하게 저장하기
- **7~8주차:** 프로젝트 완성 & 세상에 배포하기! (🔥 우리의 최종 목표)
