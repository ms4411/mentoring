CSS Cascading (우선순위)

우선순위: 인라인 스타일 > 내부 <style> > 외부 CSS > 브라우저 기본

❶ 인라인 스타일 (style="color:red") → 빨간색이 이김!

❷ 외부 CSS(파랑) vs 내부 style(파랑) → 내부가 이김

❸ 내부 스타일만 적용 → 초록색
